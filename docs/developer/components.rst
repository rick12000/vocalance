Overview
--------

The audio system documentation covers:

- **System Architecture**: High-level overview of the complete audio processing pipeline
- **Recording System**: Detailed documentation of audio capture and Voice Activity Detection
- **Speech-to-Text**: Comprehensive guide to the dual-engine STT system
- **Dictation System**: Complete documentation of dictation modes and AI enhancement
- **Sound Recognition**: Technical details of the machine learning-based sound classification
- **Configuration Reference**: Complete parameter reference for all audio settings

Each document provides both macro-level architectural understanding and micro-level implementation details, including code examples, configuration parameters, performance characteristics, and troubleshooting guidance.
